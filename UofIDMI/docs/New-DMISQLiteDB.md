# New-DMISQLiteDB

## SYNOPSIS
Creates a new instance of a SQLite database locally to store the DMI information in.
This should be created automatically by the module.

## SYNTAX

```
New-DMISQLiteDB [<CommonParameters>]
```

## DESCRIPTION
Creates a new instance of a SQLite database locally to store the DMI information in.
This should be created automatically by the module.

## EXAMPLES

### EXAMPLE 1
```
New-DMISQLiteDB
```
